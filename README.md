# Customer-Churn-Prediction
This project focuses on predicting customer churn using Machine Learning techniques. The dataset contains customer details such as age, gender, contract type, and service usage.

Data preprocessing was performed by handling missing values and converting categorical data into numerical format using label encoding. A Logistic Regression model was trained to predict whether a customer will stay or leave.

The model was evaluated using accuracy score, and the project demonstrates how businesses can use data-driven approaches to improve customer retention and decision-making.
