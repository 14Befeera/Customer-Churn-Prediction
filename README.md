# Customer Churn Prediction using Machine Learning

## Description
This project focuses on predicting customer churn using machine learning techniques. It includes data preprocessing, feature encoding, and model training to identify whether a customer will stay or leave.

## Features
- Data preprocessing and cleaning
- Label encoding for categorical data
- Machine learning model implementation
- Prediction and evaluation

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Output
- Predicted customer churn
- Evaluated model using accuracy
- Gained insights into customer behavior
